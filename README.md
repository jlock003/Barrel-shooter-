# Barrel Blasters

A mobile-friendly browser target game inspired by the supplied concept image.

## Play locally

Open `index.html` in a modern browser.

## Deploy on Render

Create a **Static Site** and use:

- Branch: `main`
- Build command: leave blank
- Publish directory: `.`

## Gameplay

1. Start the practice session.
2. Tap the range to aim and fire.
3. Compensate for wind drift.
4. Score as many points as possible in 30 seconds.
